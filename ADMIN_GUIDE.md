# 管理员使用指南

## 🔐 管理员账号

系统预设了以下管理员账号名：
- `admin`
- `boss`
- `manager`

只要注册时使用以上任一用户名，即可获得管理员权限。

## 📝 创建管理员账号

1. 打开 `ai_agent_journal.html`
2. 点击「注册」标签
3. 输入管理员用户名（如：`admin` 或 `boss`）
4. 设置密码（至少6个字符）
5. 注册成功后自动登录

## 👨‍💼 管理员功能

### 1. 管理员标识
登录后，用户名旁边会显示红色的「管理员」标识。

### 2. 管理员面板
主界面顶部会显示黄色的管理员面板，包含：
- **用户选择器**：查看所有注册用户
- **统计信息**：总用户数、总日志记录数
- **导出按钮**：导出数据功能

### 3. 查看用户数据

#### 切换查看用户
1. 在管理员面板的下拉菜单中选择要查看的用户
2. 页面顶部会显示红色横幅：「管理员模式：正在查看用户 xxx 的数据」
3. 所有日志记录、周报都会切换为该用户的数据
4. 选择「选择要查看的用户...」可返回自己的视图

#### 查看内容包括
- ✅ 用户的所有日志记录
- ✅ 用户上传的图片
- ✅ 用户的周报统计
- ✅ 任务分类和时长统计

### 4. 数据导出

#### 导出单个用户数据
1. 在下拉菜单中选择要导出的用户
2. 点击「导出当前用户数据」按钮
3. 下载 JSON 文件，包含该用户的所有信息

**导出内容：**
```json
{
  "username": "用户名",
  "exportDate": "导出时间",
  "entries": [...],  // 所有日志记录
  "hasApiKey": true/false,
  "totalEntries": 数量
}
```

#### 导出所有用户数据
1. 点击「导出所有用户数据」按钮
2. 下载包含所有用户数据的 JSON 文件

**导出内容：**
```json
{
  "exportDate": "导出时间",
  "exportedBy": "管理员账号",
  "users": {
    "用户1": { ... },
    "用户2": { ... }
  }
}
```

## 💡 使用场景

### 向老板汇报
1. 登录管理员账号（如 `boss`）
2. 在下拉菜单中选择员工账号
3. 查看该员工的所有工作日志
4. 点击「导出当前用户数据」生成汇报材料

### 团队管理
1. 登录管理员账号
2. 查看管理员面板的统计信息
3. 了解团队整体使用情况
4. 点击「导出所有用户数据」获取团队数据

### 数据分析
1. 导出单个或所有用户数据
2. 使用数据分析工具分析 JSON 文件
3. 生成使用报告和趋势图表

## 🔒 权限说明

### 管理员可以
- ✅ 查看所有普通用户的数据
- ✅ 导出任意用户的数据
- ✅ 查看系统统计信息

### 管理员不能
- ❌ 修改其他用户的数据
- ❌ 删除其他用户的记录
- ❌ 查看其他管理员的数据
- ❌ 添加或删除日志（只能查看）

## 📊 导出数据格式

### 日志记录结构
```json
{
  "id": 时间戳,
  "date": "2026-01-30",
  "category": "编码",
  "task": "任务标题",
  "description": "详细描述",
  "duration": 60,
  "images": ["base64图片数据..."]
}
```

### 用途示例
- 生成 Excel 报表
- 制作数据可视化图表
- 导入其他分析工具
- 备份团队数据

## ⚠️ 注意事项

1. **隐私保护**：管理员应妥善保管查看到的数据，尊重员工隐私
2. **权限限制**：管理员只能查看，不能修改他人数据
3. **账号安全**：管理员账号密码应设置得足够复杂
4. **数据安全**：导出的数据文件包含敏感信息，请妥善保管

## 🚀 快速开始

```
1. 注册管理员账号（用户名：admin）
2. 登录后查看管理员面板
3. 选择要查看的用户
4. 导出数据用于汇报
```
