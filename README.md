# AI Agent 使用日志

一个优雅的 AI Agent 使用日志记录系统，支持多用户账号，帮助你追踪和总结每天使用 AI 工具的情况。

## ✨ 功能特性

### 👤 账号系统（v1.9 新增）
- **用户注册/登录**：支持多账号独立使用
- **数据隔离**：每个用户的日志、图片、API密钥完全独立
- **安全认证**：密码加密存储，登录状态持久化
- **用户管理**：显示当前用户，支持安全退出
- **无缝切换**：支持多用户登录切换，数据自动加载

### 📝 日志记录
- **智能标题提取**：集成 Claude API，自动从详细内容中提取任务标题
- **多种任务类型**：编码、调试、学习、文档、数据分析、创意设计等
- **时长追踪**：记录每个任务的使用时长
- **图片附件**：支持上传多张图片，记录更直观（v1.8 新增）

### 🖼️ 图片功能
- **多图上传**：一次可选择多张图片
- **实时预览**：上传前预览图片，支持单独删除
- **大图查看**：点击图片查看全屏大图
- **本地存储**：图片转为 base64 本地存储，保护隐私
- **大小限制**：单张图片限制 5MB

### 📊 周报生成
- **自动统计**：任务数、总时长、任务分类
- **每日详情**：按日期分组显示所有任务
- **智能分析**：主要任务类型、平均使用时长
- **多周选择**：查看历史周报

### 💾 数据管理
- **本地存储**：使用 localStorage 持久化保存
- **用户隔离**：每个用户数据完全独立
- **数据导出**：支持导出 JSON 格式数据
- **周报导出**：导出纯文本格式周报

### 🎨 设计特点
- **苹果风格**：采用 Apple Design 设计语言
- **响应式**：完美适配桌面和移动端
- **流畅动画**：毛玻璃效果、渐变、动画过渡
- **暗色背景**：专业汽车摄影背景

## 🚀 使用方法

### 首次使用
1. 在浏览器中打开 `ai_agent_journal.html`
2. 点击「注册」标签
3. 输入用户名（3-20个字符）和密码（至少6个字符）
4. 注册成功后自动登录

### 登录使用
1. 在登录页面输入用户名和密码
2. 点击「登录」进入主界面
3. 开始记录你的 AI 使用情况

### 配置 AI 功能（可选）
1. 点击「AI智能提取」区域的「配置」按钮
2. 输入你的 Claude API 密钥（格式：`sk-ant-api...`）
3. 点击「保存」
4. 现在可以使用「生成标题」按钮自动提取任务标题

### 添加记录
1. 选择日期、任务类型、输入时长
2. 在「详细内容」中描述任务详情
3. 点击「生成标题」自动生成标题，或手动输入
4. （可选）点击「添加图片」上传相关截图
5. 点击「保存记录」

### 查看周报
1. 切换到「周报」标签
2. 选择要查看的周
3. 查看统计数据和详细任务列表
4. 点击「导出周报」下载文本格式

### 退出登录
点击右上角的「退出登录」按钮即可安全退出

## 🔒 隐私说明

- 所有数据（包括用户账号、图片、日志）仅保存在浏览器本地 localStorage
- 密码使用 Base64 编码存储在本地
- 每个用户的数据完全隔离，互不干扰
- Claude API 密钥按用户独立存储，不会上传到任何服务器
- 不收集任何用户数据
- 图片转为 base64 格式本地存储

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端服务器
- 使用 localStorage 持久化和数据隔离
- Claude API（可选）用于标题提取
- FileReader API 用于图片处理
- Base64 编码用于密码加密

## 💡 使用建议

1. **定期导出数据**：虽然使用 localStorage 持久化，但建议定期导出数据备份
2. **密码安全**：请设置足够复杂的密码（至少6个字符）
3. **图片大小**：单张图片限制5MB，建议压缩后上传
4. **清除数据**：清除浏览器数据会删除所有账号和记录，请谨慎操作
